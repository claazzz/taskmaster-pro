# 🚀 TaskMaster Pro

Сервис управления задачами с аналитикой, JWT-авторизацией и Docker-контейнеризацией.

## ✨ Возможности
- Регистрация и JWT-авторизация
- CRUD операции с задачами
- Приоритеты задач (low/medium/high)
- Аналитика: статистика по задачам, активность по дням
- Docker-контейнеризация (PostgreSQL + Redis + FastAPI)
- Автоматическая документация API (/docs)

## 🛠️ Технологии
- **FastAPI** — современный фреймворк для API
- **PostgreSQL** — основная база данных
- **Redis** — кэширование
- **SQLAlchemy** — ORM
- **Alembic** — миграции
- **JWT** — авторизация
- **Docker** — контейнеризация
- **pytest** — тестирование
